# 🧭 Step By Step Guide

## 🚀 Installation

Let's start by cloning the repo locally

```sh
git clone git@github.com:na6im/park-depot-Nassim.git
```

install the dependencies

- `yarn`

## 🧼 Ready to run

- `yarn start`
